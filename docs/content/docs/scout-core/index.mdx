Scout core provides easy routines for managing and updating scout state in your web applications.

## Key Features
- **Real-time Updates**: Supabase-powered real-time data synchronization
- **State Management**: Redux-based state management with loading states
- **Proactive Cacheing**: Load data from local storage and refresh with network for snappy loading
- **Team Management**: Swap between teams and roles
